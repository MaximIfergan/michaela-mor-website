# How to Add a New Exhibition

Hi Michaela! This guide explains how to prepare your exhibition data so Maxim can add it to your website.

---

## What You Need to Provide

### 1. Exhibition Info
- **Exhibition Name**: (e.g., "Bezalel Academy")
- **Year**: (e.g., 2025)
- **Location/Venue**: (optional)

### 2. Images
- A folder with all the exhibition images
- **Important**: Choose which image should be the **cover image** (thumbnail shown on the Works page)
- Recommended: Name images in the order you want them displayed (e.g., 01.jpg, 02.jpg, etc.)

### 3. Artwork Details (for each image)
Fill out the table below for each image:

## How to Send

1. Create a folder with the exhibition name (e.g., "Bezalel Academy 2025")
2. Put all images inside
3. Include a text file with the filled template above
4. Send the folder to Maxim (via Google Drive, WeTransfer, or any file sharing)

That's it!
